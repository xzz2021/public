**基本数据类型(4类):**

1. 整数类型: byte short int long, 整数默认为int
2. 浮点类型: float double
3. 字符类型: char ( 其本质是unicode编码,可以当作int类型)
4. 布尔类型: boolean

**引用数据类型:**

1. 类
2. 接口
3. 数组

**常量:** final

| 数据类型 | 长度  | 范围                |
| -------- | ----- | ------------------- |
| byte     | 1字节 | -128 ~ 127          |
| short    | 2字节 | -32768 ~ 32767      |
| int      | 4字节 | 正负21亿            |
| long     | 8字节 | -2^63 ~ 2^63 -1     |
| float    | 4字节 | 单精度: 小数点后7位 |
| double   | 8字节 | 双精度: float双倍   |
| char     | 2字节 | -128 ~ 127          |
| boolean  | 1位   | true false          |
